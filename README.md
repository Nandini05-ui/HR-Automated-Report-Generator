# HR-Automated-Report-Generator
Excel-based HR project for employee attendance and salary automation

## Project Overview
This Excel-based project automates HR attendance and salary management.  
It calculates employee salaries, tracks attendance, and generates automated department-wise reports.

## Sheets Used
1. **Title & Table of Contents** – Shows project name and structure for easy navigation.  
2. **Employee Details** – Stores employee info (ID, Name, Department) as master data.  
3. **Departments** – Lists all departments (HR, IT, Finance, Marketing, Sales) for department-wise reports.  
4. **Monthly Attendance (Jan–Dec)** – Marks Present, Absent, and Leave days for automatic monthly attendance calculation.  
5. **Leave Tracker** – Tracks total leaves and balances, linked to attendance and salary.  
6. **Department-wise Salary** – Calculates Basic, Allowances, Deductions, PF, Gross & Net Salary using Excel formulas.  
7. **Salary Tracker** – Consolidates salaries from all departments for easy comparison.  
8. **Automated Reports** – Generates automatic HR reports using formulas, labels, data validation, and hyperlinks for quick navigation.

## Tools & Techniques
- Microsoft Excel  
- Excel Formulas: SUM, IF, VLOOKUP/XLOOKUP  
- Data Validation for accurate inputs  
- Labels & Hyperlinks for easy navigation  
- Automated calculations and report generation  

## Project Outcome
- Fully automated HR attendance and salary system  
- Reduced human errors in payroll  
- Department-wise and monthly analysis made easy  
- Professional academic-level HR project ready for real-world scenarios
